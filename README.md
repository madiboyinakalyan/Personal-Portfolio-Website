# 🌐 Personal Portfolio Website

## 📌 Overview

This project is a responsive personal portfolio website developed as part of the Microsoft Internship Program. It showcases my technical skills, projects, certifications, and contact information through a clean and modern user interface.

The portfolio is designed to provide visitors with an overview of my background, technical expertise, and project experience while demonstrating fundamental front-end web development skills.

---

## 🚀 Features

- Responsive design
- Professional profile section
- Skills showcase
- Project portfolio
- Certifications section
- Contact information
- Clean and modern UI
